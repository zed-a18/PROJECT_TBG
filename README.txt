TEXT BASED ADVENTURE GAME

This is the text based adventure game where the goal is to help the friendly character retrieve her lost slipper.

Once you run the Python file you will be given actions to input to proceed. 

You will find items that will be added to your inventory, talk to friendly npc, use items from inventory to either fight or bribe and open a locked door.

Once you find the lost slipper the game will end.
